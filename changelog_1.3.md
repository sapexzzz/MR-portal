# MR Portal 1.3

Дата: 2026-03-22

## Изменено

- Версия мода обновлена до 1.3.0
- После входа в портал он больше не исчезает мгновенно
- По умолчанию портал держится 3 секунды после телепорта и только затем начинает плавно схлопываться
- Добавлен параметр portalCloseDelayTicks для управления этой задержкой
- Искра переведена полностью на белые частицы
- Кнопка телепорта теперь автоматически отключается, если ресурсов не хватает
- Tooltip обычного посоха теперь показывает Заряжено вместо нулевого значения перезарядки
- GUI переработан для более аккуратной компоновки без выхода текста за границы панели

## Добавлено

- Возможность задать собственное имя новой точки перед сохранением
- Постоянный enchanted-glint для обоих посохов
- Белые частицы вокруг посохов при ношении
- Новый changelog версии 1.3
