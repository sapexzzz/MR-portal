# MR Portal 1.2

Дата: 2026-03-22

## Изменено

- Версия мода обновлена до 1.2.0
- Обычный посох теперь использует фиксированную стоимость телепорта вместо расчета по расстоянию
- Стоимость телепорта обычным посохом вынесена в конфиг через portalPearlCost
- Добавлена возможность отключать слепоту после телепорта через applyBlindnessOnTeleport
- GUI переработан и стал нагляднее: выбранная точка, ресурсы и статус действия теперь видны сразу
- Искра предпросмотра появляется сразу после нажатия ПКМ по посоху и исчезает при закрытии экрана без выбора
- Искра получила новый particle-эффект
- Схлопывание портала после телепорта сохранено и синхронизировано с новой искрой

## Добавлено

- Новый changelog для версии 1.2
- Обновленные README на русском и английском
