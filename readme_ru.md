# MR Portal

MR Portal — мод для Fabric 1.20.1, который добавляет систему персональных порталов и вейпоинтов в стиле Magic Rampage.

Текущая версия: 1.9.0

## Возможности

- Посох портала для выживания с рецептом крафта
- Бесконечный посох портала для креатива
- Персональные точки телепортации по UUID игрока
- Отдельная вкладка предметов MR Portal
- GUI без паузы игры в одиночном мире
- Отдельный режим обычного и креативного посоха
- Искра появляется сразу при открытии меню выбора точки
- Новый particle-эффект искры вместо прежнего вида
- Искра теперь использует только белые частицы
- Черный портал, который раскрывается из искры и затем схлопывается обратно в точку
- Портал появляется перед игроком, а не под ним
- После телепорта портал живет еще заданное время и только потом плавно схлопывается
- Настраиваемый эффект слепоты при переносе, который можно отключить
- Настройки таймингов и геометрии через config/mr_portal.json
- Улучшенный и более понятный GUI со статусом ресурсов
- Возможность задавать собственное имя новой точки
- Переименование существующих точек прямо из GUI
- Точки группируются по измерениям с заголовками
- Имя точки запрашивается через диалог только при добавлении
- Портал всегда плавно схлопывается даже если в него никто не зашел
- Посохи всегда выглядят зачарованными и дают белые частицы при ношении
- В tooltip обычного посоха показывается состояние перезарядки
- Портал назначения теперь появляется даже при телепортации за пределы дальности прорисовки
- Поддержка Cloth Config API для редактирования настроек прямо в игре через ModMenu
- Настройка языка мода (auto / en / ru) в экране конфигурации
- Мобы и другие игроки могут телепортироваться через портал
- Искра предпросмотра (при выборе точки) видна другим игрокам поблизости
- Свиток телепорта — одноразовый расходуемый предмет для телепортации (выглядит как зачарованная бумага)
- Клавиша быстрого открытия GUI (V по умолчанию) — работает если в инвентаре есть посох или свиток
- Блокировка телепорта между измерениями для обычного посоха и свитка (бесконечный посох не ограничен)
- Портал появляется на расстоянии 2 блоков перед игроком и стоит на уровне земли
- Рецепты автоматически разблокируются при получении ингредиентов
- Настраиваемое вертикальное смещение портала (portalVerticalOffset)
- Настраиваемая скорость анимации открытия/закрытия портала (portalAnimationSpeed)
- Исправлена настройка языка — теперь изменяет только тексты мода, не затрагивая интерфейс Minecraft
- Исправлен слишком большой хитбокс портала: телепорт срабатывает только при фактическом входе в плоскость портала
- Можно выбрать одну метку как избранную
- Добавлен второй настраиваемый бинд для мгновенного открытия портала к избранной точке (по умолчанию клавиша ё)

## Предметы

### Portal Staff

- ID: mr_portal:portal_staff
- Используется в выживании
- Имеет обычный кулдаун
- Тратит фиксированное количество эндер-жемчуга за телепорт

### Teleport Scroll

- ID: mr_portal:teleport_scroll
- Одноразовый свиток телепортации
- Стакается до 16 штук
- Выглядит как зачарованная бумага
- Не требует эндер-жемчуга
- Расходуется при телепортации
- Не работает для телепортации между измерениями

### Infinite Portal Staff

- ID: mr_portal:infinite_portal_staff
- Без кулдауна
- Не тратит эндер-жемчуг
- Предназначен для креатива и админского использования

## Рецепты

### Портал-посох

- углы: обсидиан
- стороны: око Эндера
- центр: звезда Незера

### Свиток телепорта

- стороны: эндер-жемчуг (4 шт.)
- центр: бумага
- выход: 2 свитка телепорта

## Как использовать

1. Возьми посох портала или свиток телепорта в руку.
2. Нажми ПКМ, чтобы открыть список точек и увидеть предпросмотр искры.
3. Добавь текущую точку или выбери уже существующую.
4. При выборе точки портал откроется перед игроком.
5. В точке назначения одновременно появится искра и портал.
6. Зайди в портал, чтобы телепортироваться.

Также можно нажать клавишу V (настраивается в управлении), чтобы открыть меню без ПКМ, если в инвентаре есть посох или свиток.

## Конфиг

Файл создается автоматически:

- config/mr_portal.json

Основные параметры:

- cooldownSeconds: кулдаун обычного посоха в секундах
- maxWaypoints: лимит точек на игрока
- portalOpenDelay: время раскрытия портала в тиках
- portalActiveTicks: сколько тиков портал остается открытым после раскрытия
- portalCloseDelayTicks: сколько тиков портал остается после телепорта до начала схлопывания
- portalPearlCost: стоимость одного телепорта обычным посохом
- applyBlindnessOnTeleport: включает или выключает слепоту после телепорта
- blindnessDuration: длительность слепоты в тиках
- portalScale: размер черного портала
- portalSpawnDistance: дистанция появления портала перед игроком
- portalVerticalOffset: вертикальное смещение портала (от -2.0 до 5.0)
- portalAnimationSpeed: множитель скорости анимации портала (от 0.1 до 5.0)

Настройки можно редактировать прямо в игре через ModMenu → MR Portal (требуется Cloth Config API).

Значения по умолчанию в 1.9.0:

- cooldownSeconds = 60
- maxWaypoints = 10
- portalOpenDelay = 60
- portalActiveTicks = 100
- portalCloseDelayTicks = 60
- portalPearlCost = 3
- applyBlindnessOnTeleport = true
- blindnessDuration = 40
- portalScale = 1.5
- language = auto
- portalSpawnDistance = 2.2
- portalVerticalOffset = 0.0
- portalAnimationSpeed = 1.0

## Избранная метка и быстрый портал

- В интерфейсе можно отметить одну точку как избранную
- Избранная точка помечается звездой и сохраняется на сервере
- Новый быстрый бинд по умолчанию использует клавишу ё
- При быстром открытии приоритет такой: бесконечный посох, затем обычный посох, затем свиток телепорта
- Для быстрого перехода в другое измерение нужен бесконечный посох или права креатива

## Сборка

Требования:

- Java 17+ для проекта
- В текущем окружении сборка была проверена через Java 21

Команда:

```bash
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk
export PATH=$JAVA_HOME/bin:$PATH
./gradlew build
```

Готовый jar появляется в build/libs.
