# Changelog 1.4.0

## GUI

- Полностью переработан интерфейс выбора точек
- Точки теперь группируются по измерениям с заголовками секций
- Текущее измерение игрока всегда отображается первым в списке
- Имя точки запрашивается через модальный диалог только при добавлении новой точки
- Убрано поле ввода имени из основного экрана — оно больше не мешает
- Добавлена кнопка «Имя» для переименования уже существующих точек
- Улучшен выбор точек — кликабельные строки с правильным позиционированием
- Добавлен скроллбар для длинных списков точек
- Диалоги подтверждения (удаление, ввод имени, переименование) перекрывают интерфейс полупрозрачным фоном

## Портал

- Портал теперь всегда плавно схлопывается в конце жизни, даже если в него никто не заходил
- Сервер начинает закрытие после окончания активного периода (portalOpenDelay + portalActiveTicks), затем выжидает portalCloseDelayTicks перед отправкой PORTAL_CLOSE
- Клиентский эффект портала получает увеличенный lifetime (+40 тиков) для гарантии плавного закрытия по серверной команде

## Сеть

- Добавлен пакет RENAME_WAYPOINT_C2S для переименования точек
- Сервер нормализует имя при переименовании (trim, ограничение 32 символа, fallback на автоимя)

## Хранилище

- ServerWaypointStore: добавлен метод renameWaypoint для обновления имени точки по UUID

## Локализация

- Добавлены ключи: screen.mr_portal.rename, screen.mr_portal.rename_title, screen.mr_portal.dialog_ok, screen.mr_portal.dialog_cancel
- Обновлены оба файла локализации (ru_ru.json, en_us.json)

## Прочее

- Версия обновлена до 1.4.0
- Обновлены readme.md и readme_en.md
